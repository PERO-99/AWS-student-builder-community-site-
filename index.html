01 // About the Event

One campus. One day.
A full cloud curriculum.

Student Builder Community Day brings the AWS Cloud Club at Rungta together with builders from across Chhattisgarh for a day of real workshops — not keynote-only sessions.

Whether you've never opened the AWS console or you're already shipping on Lambda, there's a track built exactly for where you are.

  • Hands-on AWS Labs & Real Cloud Workshops
  • AI/ML Sessions with Amazon Bedrock & SageMaker
  • Live Build Sprint — ship something by 5 PM
  • Certificate + AWS Cloud Club swag for every builder
